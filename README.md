# About
Free simple crypto-shop source. Based on public API`s: Blockchain, Ethplorer, XMRChain and CryptoID.

# Features

  - No need API key`s
  - Clear PHP code
  - Simple admin panel
  - Good design)
> But only support text items

### Instruction
1. Download
2. Set `chmod 777 install.php`
3. Run `install.php` and set you data
